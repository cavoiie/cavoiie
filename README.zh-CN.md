# CFD、PyFluent 与 AI 辅助仿真工作流

[English](README.md) | 简体中文

我关注 Ansys Fluent、PyFluent 和 agent 辅助仿真工具链中的实用 CFD 工作流。

当前重点是让仿真设置、求解器自动化、收敛检查和验证步骤更明确、更可复现。

## Featured Project

### [Fluent CFD Skill](https://github.com/cavoiie/fluent-cfd-skill)

一个面向 Ansys Fluent 和 PyFluent CFD 工作流的 Codex skill。它提供紧凑的工作流层，用于：

- Fluent case 设置与审查
- 边界条件和网格质量检查
- 湍流模型和近壁处理决策
- 基于残差、监控量和守恒检查的收敛评估
- 面向本地 Fluent 自动化的 PyFluent MCP 工具调用规范

## Tooling

- Ansys Fluent 和 PyFluent
- Model Context Protocol (MCP)
- 面向仿真工作流的 Python 自动化
- 用于复用工程流程的 Codex skills

## Engineering Interests
